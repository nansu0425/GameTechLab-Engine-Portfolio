# GameTechLab Engine Portfolio

> **크래프톤 정글 게임테크랩 2기** 과정에서 개발했던 게임 엔진 포트폴리오

게임 엔진의 **동작 원리**를 탐구하기 위해, **Unreal Engine** 아키텍처를 참고해서 게임 엔진을 개발했습니다. 주마다 **새롭게 구성되는 팀**에서 주차별 과제를 수행하며 엔진을 발전시켰습니다. 개발했던 엔진으로 **세 번의 게임잼**에서 게임도 만들었습니다.

## About Project

- **2025.09.02 – 2025.12.11**, 14주간 진행
- **매 주차 3~4명 팀**으로 진행, 팀원은 매 주차 랜덤 변경
- **시작 코드베이스도 매 주차 변경** → 팀원 중 한 명의 전 주차 코드베이스를 골라 그 위에서 시작
- **매 주차 새 과제** → 1주 안에 과제 수행 후, 다음 주에는 다시 새 팀·새 코드베이스로 이동

주차별 리포지토리와 **제가 담당한 작업**은 [주차별 작업](docs/weekly_work.md) 에서 확인할 수 있습니다.
